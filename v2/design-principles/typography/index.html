LINE HEIGHTS

--line-height-tight: 1.2

Tight line height is perfect for large headlines and display text where you want a compact, impactful appearance. Best used sparingly for maximum effect.

--line-height-normal: 1.6

Normal line height provides comfortable reading for body text and general content. This is the default for most text elements in the design system.

--line-height-relaxed: 1.8

Relaxed line height offers extra breathing room, ideal for longer passages of text or when you want to create a more open, premium feel in your content.
